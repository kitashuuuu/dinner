# サラダを作る
## 材料
- [x] 鳥ミンチを買う
- [x] ドレッシングを買う
- [x] サラダを買う
## 手順
1. **食材を水洗いする**  
   野菜を全部水洗いする
2. **食材を切る**  
   食材を切る
3. **食材を混ぜる**  
   切った食材を一つにして混ぜる
4. **ドレッシングをかける**  
   胡麻ドレッシングをかけて完成
