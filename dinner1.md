##材料

 - [x] レタス 1個
 - [x] 鶏ミンチ 100g
 - [x] ごま油 大さじ1
 - [x] 片栗粉 大さじ1
 - [x] 糸唐辛子 適量
 - [x] オイスターソース 大さじ2
 - [x] しょうゆ 大さじ1/2
 - [x] こしょう 少々
 - [x] 水　1カップ
###調理工程

 - [x] レタスを切る
 - [x] フライパンを熱する
 - [x] レタスを炒める
 - [x] ミンチに味付けをする
 - [x] ミンチを炒める
 - [x] 片栗粉を加えてとろみをつける
 - [x] 糸唐辛子を乗っける
